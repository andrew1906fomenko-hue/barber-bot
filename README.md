# Telegram Web App для записи к барберу

Готовый проект бота с красивым Web App интерфейсом:
- Постоянные кнопки в боте: **Запись**, **Услуги**, **Наш адрес**.
- Web App календарь на 14 дней вперёд (дни недели + реальные даты).
- Выбор свободного времени и отправка записи.
- После записи данные уходят администратору по Telegram ID.
- Админ получает обновленный календарь записей и может открыть его командой `/calendar`.

## Запуск

1. Установите зависимости:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2. Создайте `.env`:

```env
BOT_TOKEN=ваш_токен_бота
ADMIN_CHAT_ID=ваш_telegram_id
WEB_APP_URL=https://ваш-домен
```

> Для локальной разработки можно временно указать `WEB_APP_URL=http://localhost:8000`.

3. Запустите:

```bash
uvicorn app.main:api --host 0.0.0.0 --port 8000
```

## Что проверить

- `/start` показывает кнопки и приветствие.
- Кнопка `🗓 Запись` открывает Web App.
- Календарь показывает 14 дней.
- После записи админ получает сообщение + календарь.
- `/calendar` у админа показывает записи на ближайшие 14 дней.
